## 前置知识

Eyestack 框架的文档假设你已经掌握了以下知识：

1. 良好的 C++ 基础，尤其是 Mordern C++ 的新特性

   https://zh.cppreference.com

2. CMake 构建系统的使用、C++ 项目编译构建的全过程

   https://cmake.org/documentation/

3. Qt 5.15 框架的使用

   https://doc.qt.io/qt-5.15/reference-overview.html

   重点是 Qt Widgets 模块和 QThread 并发编程。

   如何与 CMake 结合：

   https://doc.qt.io/qt-5.15/cmake-manual.html

4. Qt Creator IDE 开发工具

   https://doc.qt.io/qtcreator/index.html

5. Boost.Test 单元测试框架的使用

   https://www.boost.org/doc/libs/1_76_0/libs/test/doc/html/index.html

6. Python3 编程

   https://docs.python.org/3/

7. Git 和远程仓库的使用

   https://git-scm.com/doc

8. Doxygen 文档生成工具的使用

   http://www.doxygen.nl/

9. （推荐了解）Win32 应用开发和 UCRT

   https://docs.microsoft.com/en-us/windows/win32/desktop-programming

   https://docs.microsoft.com/en-us/cpp/c-runtime-library/c-run-time-library-reference
